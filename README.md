# Sales Performance Dashboard 📊

Projeto desenvolvido utilizando Power BI com objetivo de analisar desempenho de vendas, lucro e indicadores comerciais.

## Objetivo

Criar um dashboard interativo capaz de responder perguntas de negócio:

- Qual a receita total?
- Qual o lucro obtido?
- Qual a margem de lucro?
- Quais produtos possuem maior venda?
- Quais países apresentam maior desempenho?
- Quais segmentos geram mais lucro?

## Ferramentas utilizadas

- Microsoft Power BI
- Power Query
- DAX
- Excel

## Indicadores criados

### KPIs

- Receita Total
- Lucro Total
- Margem de Lucro
- Quantidade de Produtos

### Análises

- Vendas por Produto
- Vendas por País
- Lucro por Segmento

## Processo

1. Importação dos dados no Power BI.
2. Tratamento dos dados utilizando Power Query.
3. Criação de medidas utilizando DAX.
4. Construção do dashboard.
5. Análise dos resultados.
