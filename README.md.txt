# Sales Performance Dashboard 📊

Projeto desenvolvido utilizando Power BI com o objetivo de analisar desempenho de vendas, lucro e indicadores comerciais através de um dashboard interativo.

## Objetivo

Criar um dashboard capaz de responder perguntas de negócio relacionadas ao desempenho de vendas:

- Qual a receita total?
- Qual o lucro obtido?
- Qual a margem de lucro?
- Quais produtos apresentam maior volume de vendas?
- Quais países possuem melhor desempenho?
- Quais segmentos geram maior lucro?

## Fonte dos Dados

Foi utilizado um conjunto de dados de amostra (Sample Dataset) disponibilizado para treinamento e desenvolvimento de análises no Power BI.

Os dados foram utilizados para simular um cenário de análise comercial, contendo informações como:

- Produtos
- Países
- Segmentos
- Vendas
- Lucro
- Ano

## Ferramentas utilizadas

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)

## Processo de desenvolvimento

1. Importação do conjunto de dados de amostra no Power BI.
2. Tratamento e transformação dos dados utilizando Power Query.
3. Criação de medidas utilizando DAX.
4. Desenvolvimento dos indicadores de desempenho (KPIs).
5. Construção do dashboard para análise dos resultados.

## Indicadores criados

### KPIs

- Receita Total
- Lucro Total
- Margem de Lucro
- Quantidade de Produtos

### Análises realizadas

- Vendas por Produto
- Vendas por País
- Lucro por Segmento
